HTML Semantic Elements

HTML Semantic Elements
Element Name Short Description
<header> Header Element Represents introductory content, typically a group of introductory or navigational aids
<nav> Navigation Element Represents a section of a page whose purpose is to provide navigation links, either within or to other documents
<main> Main Element Represents the dominant content of the <body> of a document
<section> Section Element Represents a generic standalone section of a document, which doesn't have a more specific semantic element to represent it. Sections should always have a heading, with very few exceptions.
<article> Article Element Represents a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable (e.g., in syndication). Examples include: a forum post, a magazine or newspaper article, or a blog entry, a product card, a user-submitted comment, an interactive widget or gadget, or any other independent item of content
<aside> Aside Element Represents a portion of a document whose content is only indirectly related to the document's main content. Asides are frequently presented as sidebars or call-out boxes
<footer> Footer Element A <footer> typically contains information about the author of the section, copyright data or links to related documents

HTML Table Elements

Table Elements
Element Name Short Description
<table> Table Represents tabular data in table comprised of rows and columns of cells containing data
<thead> Table Head A set of rows defining the head of the columns of the table
<tbody> Table Body Encapsulates a set of table rows indicating that they comprise the body of the table
<tfoot> Table Foot Defines a set of rows summarizing the columns of the table
<tr> Table Row Defines a row of cells in a table, incl. <td> (data cell) and <th> (header cell)
<th> Table Header Defines a cell as the header of a group of table cells
<td> Table Data Defines a cell that contains data

HTML Table Attributes

Table Attributes
Attribute Name Short Description
colspan="2" Column Span Contains an integer value that indicates how many columns a cell extends e.g. 2
rowspan="4" Row Span Contains an integer value that indicates how many rows a cell extends e.g. 4
scope="col" Header Scope: Column Defines the cells that the header (defined in the <th>) element relates to all cells of the column it belongs to
scope="row" Header Scope: Row Defines the cells that the header (defined in the <th>) element relates to all cells of the row it belongs to
